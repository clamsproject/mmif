CLAMS Exchange Vocabulary

version 1.0

The CLAMS Services Exchange Vocabulary defines an ontology of terms for a core of objects and features exchanged amongst tools that process multi-media data. It is based on the LAPPS Web Service Exchange Vocabulary at http://vocab.lappsgrid.org. The vocabulary is being developed bottom-up on an as-needed basis for use in the development of the CLAMS platform.
