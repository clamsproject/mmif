Annotation > Region


Definition An annotation over a region in primary data where primary data can be a text, an image, an audio stream or a video streem. Typically one of the sub types of this will be used.
URI http://mmif.clams.ai/vocabulary/1.0/Region.html

Metadata

Property Type Description
unit String The unit used to determine boundaries. For text regions this is typically *offsets* to indicate character offsets, for images this is could be *coordinates*, and for audio or video streams *seconds*, *milliseconds*, *timestamps* or *frames*.
