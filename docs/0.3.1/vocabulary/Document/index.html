Thing > Document


Definition A document of some media type. Annotations directly or indirectly anchor to documents. In CLAMS, a document typically refers to an external file fia the *location* property, but for text documents the actual content can be in-line in the document type.
URI http://mmif.clams.ai/0.3.1/vocabulary/Document
